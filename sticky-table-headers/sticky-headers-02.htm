Borders Continued

This step adds a container element, a DIV. This container limits the visible height of the table inside, and provides scroll-bars if the table size exceeds its container. The container element also provides a border, so the table cells no longer have to provide top and left border lines. I kept the right border of the right-most cells, this gives a nice visual separation to the vertical scroll-bar.

.sticky-table-container {
	border : 1px solid silver ;
	display : inline-block ;
	width : auto ;
	height : 100px ;
	overflow-y : auto ;
	}
