Adding a Sticky Header Column

One additional rule makes the cells in the first table row sticky:

table.sticky-headers tr :first-child {
	position : sticky ;
	left : 0px ;
	background-color : white ;
	}
table.sticky-headers :first-child tr:first-child :first-child {
	z-index : 1 ;
	}

A background color is required to prevent the data cells from shining through.
