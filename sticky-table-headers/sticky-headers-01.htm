DIY Table Clone Test 1: Borders

The style setting border-collpase:collapse on the table, combined with something like border : 1px solid silver on table cells should give a nice one-pixel border around table cells. While Firefox does it right, Chrome and Edge spoiled it: The border thickness varies unpleasantly at regular intervals. This first step addresses that with border-collapse:separate (which is the dafault) and border-spacing:0 addresses this flaw. To obtain a one-pixel border, table cell will have a border line at the bottom and right side:

table.sticky-headers {
	border-spacing : 0 ;
	}
table.sticky-headers tr * {
	border-color : #CCC ;
	border-width : 1px ;
	border-bottom-style : solid ;
	border-right-style : solid ;
	}

Every first cell in a row needs an additional left border:

table.sticky-headers :first-child tr * {
	border-top-style : solid ;
	}

The cells in the first row get an additional top border.

table.sticky-headers tr :first-child {
	border-left-style : solid ;
	}
