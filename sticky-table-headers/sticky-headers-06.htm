Sticky Row-Group Headers with Overflow

This variant uses a row-group header cell that spans a single column, followed by an empty TD cell that spans the rest of the columns. The TH cell text can overflow into the TD cell on the right.

<thead><tr class="rowgroup-header">
	<th colspan="1">Group 1</th>
	<td colspan="9"></td>
</tr></thead>

The stylesheet code for the TH cell gets the overflow permission:

table.sticky-headers .rowgroup-header th {
	border-right-style : none ;
	overflow-x : visible ;
	}

Because the row-group header text now determines the width of the first table column, we neet to set the column width explicitly:

table.sticky-headers tr :first-child {
	…
	max-width : 100px ;
	}
