Counter Component

By the way: The listings in this document are enumerated with this technique.

Example: If you add the figure class name to a FIGCAPTION element, the figure caption will be automatically enumerated:

<figcaption class="figure"></figcaption>
<img src="/inc/collapsible/orange-plus.svg"/>
<figcaption class="figure"></figcaption>
<img src="/inc/collapsible/orange-minus.svg"/>
<figcaption class="figure"></figcaption>
<img src="/inc/collapsible/orange-dot.svg"/>

… you will get the figcaption elements enumerated:

Now, how does that work?

The CSS counter component consists of a single CSS file, and is imported in page.css:

@import url( colors.css );
@import url( header-2.css );
@import url( footer-2.css );
@import url( toolbar/toolbar-2.css );
@import url( collapsible/collapsible-5.css );
@import url( navigation/navigation-3.css );
@import url( counter.css );

The content of the file is rather simple. The first instruction defines a counter reset on the BODY element for three different counters:

body { counter-reset : listing-counter figure-counter table-counter ; }

The following rule increments the listing counter on every figcaption element of class listing, and generates the formatted counter value in front of the element's text content:

figcaption.listing:before { 
	counter-increment : listing-counter ; 
	content : "Listing " counter( listing-counter ) ": " ; 
	}

If you create this HTML code, …

<figcaption class="listing">The code to be demonstrated</figcaption>

… you will get the following listing title:

Listing 1: The code to be demonstrated

Likewise, the following rules generate text content for figcaption elements of class figure, table and table captions of class autonum:

figcaption.figure:before { 
	counter-increment : figure-counter; 
	content : "Figure " counter(figure-counter) ": " ; 
	}

figcaption.table:before { 
	counter-increment : table-counter; 
	content : "Table " counter(table-counter) ": " ; 
	}

table > caption.autonum:before { 
	counter-increment : table-counter ; 
	content : "Table " counter(table-counter) ": " ; 
	}
