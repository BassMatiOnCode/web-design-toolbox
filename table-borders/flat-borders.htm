Here is the trick:

table.flat-borders-demo-2 {
	border-spacing : 0 ;
	}

First of all, the table has the style border-collapse:separate, which is the default. Then a border-spacing:0. Then we have a one-pixel border on the right side and at the bottom of each table cell:

table.flat-borders-demo-2 tr * {
	border-color : #CCC ;
	border-width : 1px ;
	border-bottom-style : solid ;
	border-right-style : solid ;
	}

All cells in the very first row have a border at the top:

table.flat-borders-demo-2 :first-child :first-child * {
	border-top-style : solid ;
	}

And the cells in the first column have a border at the left side:

table.flat-borders-demo-2 tr :first-child {
	border-left-style : solid ;
	}

Little effort, much effect.
